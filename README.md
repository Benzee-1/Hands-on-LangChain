# Hands-on-LangChain
Examples are provided to help developping applications using LangChain libraries. Agents,  LLMs, Tools, prompt engeneering and other concepts are explained.

LangChain was first released in October 2022. The project started around that time, with its initial public versions published on GitHub. LangChain quickly gained traction in the AI and developer community as a framework for building applications with language models and tools.

We try to use the most rrecent releases, at this time when we are writing this lines, LangChain 1.0.3 is released. The LangChain 1.0.x release was a major structural update. It simplified the ecosystem, improved modularity, and cleaned up many of the breaking changes introduced in the 0.x series
